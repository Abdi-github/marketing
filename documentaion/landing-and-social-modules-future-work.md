# Landing Page And Social Post Future Work

This file tracks the polish that should remain after the current implementation pass.

## Landing Page Generator

- Promote uploaded landing-page images into a reusable media library picker with tags, previous usage, delete/replace history, and per-asset ownership metadata.
- Add screenshot-based visual QA for generated classic, balanced, and modern websites across desktop and mobile.
- Add a "regenerate design only" action that keeps copy/content but reruns designPlan, style contract, section variants, theme, and image direction.
- Enrich generated subpages with page-specific copy instead of mostly derived homepage content.
- Add per-page SEO metadata, Open Graph images, sitemap data, canonical URLs, and local-business schema.
- Add deeper industry packs: restaurant menus/allergens, clinic practitioners/services, fitness timetables, retail product offers, and service process/guarantees.
- Add named version snapshots: original AI draft, before theme change, published version, and user-created checkpoints.
- Update ADR-0027 if custom section creation is now considered accepted product scope.

## Social Post And Creative Generator

- Backfill standalone generated post images into `media_assets` so provider URLs are never the long-term durable asset URL.
- Add a media library with generated/uploaded assets, tags, reuse, edit history, and delete lifecycle.
- Add multi-slide carousel generation: cover, benefit/detail slides, proof, and CTA.
- Add a scheduling calendar, recurring campaign ideas, and Swiss holiday/event suggestions.
- Pull Meta performance metrics and feed them into next-post recommendations.
- Add approval workflows for team/regulated-business publishing.
- Add visual QA for every graphic template and aspect ratio: text overflow, contrast, missing images, and blank renders.
- Add cost/credit transparency for premium background generation and image editing.
- Add platform-specific caption variants for Facebook, Instagram, Google Business Profile, WhatsApp broadcast, and LinkedIn.
